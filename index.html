Some of the key features that makes Stream Cardano special

Multi-tenancy

Multiple users will share the same backend to minimize resource consumption and cost. That means that different user accounts need to be strictly separated.

Isolation & Security

User accounts will be kept totally separate from one another. Each project only has access to one project wallet via wallet service.

Caching

Caching results of the query for production, and share triggers for subscription over many instances of the same client application.

Access Token Lifecycle

Multiple access token lifecycle for multiple environment such as development, staging and production.

Fair Sharing & DDoS prevention

We enact the limits at different stages, to assure a size of processing per query and per dApp are within reasonable limits.

IDE

An Integrated Debugging Environment that allows you to subscribe and execute queries in real-time
